# Examples

Use these prompts to test or demonstrate `super-cloud`.

- `Use $super-cloud to design an AWS deployment for this app.`
- `Use $super-cloud to review this Terraform plan.`
- `Use $super-cloud to plan a multi-cloud migration approach.`

## Evaluation Checklist

- The skill asks only for missing context.
- The output routes to the correct module.
- The answer is actionable and specific.
- Assumptions are stated clearly.
- Risks, validation steps, or next actions are included where relevant.
